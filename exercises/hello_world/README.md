# Exercise: Hello World

Welcome to **The Gymnasium**! This exercise is designed to help you get familiar with the automated grading system.

## The Task

Implement a function `hello_world()` that takes no arguments and returns the string `"Hello, World!"`.

### Specifications

- **Function Name**: `hello_world`
- **Return Type**: `str`
- **Expected Output**: `"Hello, World!"`

### Example

In your Python code, you should be able to call the function like this:

```python
>>> hello_world()
'Hello, World!'
```

## Instructions

1. Open `exercises/hello_world/solution.py`.
2. Implement the `hello_world` function.
3. Run `make test` locally to verify your solution.
